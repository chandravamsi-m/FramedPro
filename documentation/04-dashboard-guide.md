# Dashboard Guide

The template consists of two main Dashboard views implemented as static HTML shells:

## 1. User (Client) Dashboard
Found at `pages/user-dashboard.html`. This serves as a client portal designed for corporate clients to proof, approve, bulk download, and re-order professional headshots. It consists of a collapsible sidebar and top header indicating user operations.

## 2. Admin Dashboard
Found at `pages/admin-dashboard.html`. This acts as the photographer's operational control panel, letting the photographer manage client accounts, gallery uploads, approval workflows, and invoice tracking.

> **Note:** The dashboards are frontend interfaces only lacking real backend authentication or business logic implementations. A backend language, API, or an integration tool is required to make these dashboards fully functional and secure.
