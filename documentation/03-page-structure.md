# Page Structure

- **Root files:** Contains main `index.html` (Home 1) and `home-2.html` (Home 2).
- **`/pages/`:** Holds public marketing pages (About, Services, Portfolio, Blog, Contact, FAQ, etc) and the dashboard shells (`user-dashboard.html`, `admin-dashboard.html`).
- **`/auth/`:** Contains static HTML shells for Login and Sign Up (Access Request forms).
- **`/assets/`:** Consists of folders for `css/` (custom additions and dashboard specs), `js/` (shared scripts and individual page interactions), `images/`, and `fonts/`.
- **`/components/`:** Stores reusable HTML fragments like `navbar.html` and `footer.html`. They are dynamically injected into standard public pages using JavaScript.
