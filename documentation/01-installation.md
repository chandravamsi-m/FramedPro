# Installation

1. **Extract** the ZIP file to your target directory.
2. The template uses a CDN for **Tailwind CSS**, so no `npm install` or build step is required out of the box.
3. Simply open `index.html` in your browser to view the template locally, or host it on a local server.

> **Production Note:** For optimal performance in a production environment, it is highly recommended to build Tailwind CSS properly (using the CLI with a content scan) to remove unused utility classes, rather than relying on the CDN version.
