# Customization

1. **Design Tokens:** Edit the Tailwind CSS config Block in the `<head>` of HTML files. Look for `tailwind.config = { ... }` to modify primary colors and typography.
2. **Images:** Replace placeholder images inside the `assets/images/` directory with your own assets. Making sure to keep paths identical will instantly reflect changes on the site.
3. **Icons:** Swap out SVG paths in HTML directly or bring in your own SVG icon suite.
4. **Dark Mode & RTL:** Natively supported via Tailwind classes. You can tweak dark variations by adjusting the `dark:` utility classes prefixed in the HTML blocks.
